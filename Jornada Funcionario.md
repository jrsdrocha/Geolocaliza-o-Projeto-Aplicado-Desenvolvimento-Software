Jornada Funcionario

Acessa o sistema da biblioteca
Abre o site ou sistema interno da biblioteca para iniciar a busca por um livro → pensa em resolver rapidamente a localização para continuar suas tarefas.

( Touchpoint — tela inicial de busca )

Digita o termo de busca
Insere título, autor ou assunto no campo de pesquisa do catálogo → espera encontrar rapidamente o livro desejado.

( Touchpoint — campo de busca do catálogo )

Analisa os resultados
Visualiza a lista de livros retornados pelo sistema e identifica o item correto → sente alívio por encontrar o livro rapidamente.

( Touchpoint — tela de resultados da busca )

Seleciona o livro
Clica no livro desejado para ver mais informações sobre ele → espera confirmar a localização exata dentro da biblioteca.

( Touchpoint — item da lista de resultados )

Visualiza os detalhes do livro
Observa as informações da obra como título, autor e código de chamada → sente segurança ao ver informações organizadas.

( Touchpoint — tela de detalhes do livro )

Interpreta o código de chamada
Lê o código de chamada e entende sua relação com a organização das estantes → sente que consegue se orientar melhor no acervo.

( Touchpoint — componente de código de chamada )

Abre o mapa da biblioteca
Clica na opção de visualizar a localização da obra no mapa → espera ver exatamente onde precisa ir.

( Touchpoint — botão “ver no mapa” )

Observa a localização da estante
Visualiza o mapa interno da biblioteca com destaque para o setor, corredor e estante → sente confiança de que irá direto ao local correto.

( Touchpoint — mapa digital da biblioteca )

Memoriza o local
Guarda mentalmente a localização indicada no mapa antes de ir até a estante → sente que conseguirá encontrar o livro sem dificuldade.

( Não é touchpoint direto )
